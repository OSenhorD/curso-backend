# Curso Backend

Aplicação teste para demonstrar conhecimentos básicos

## Tecnologias

- Banco de dados PostegreSQL
- ORM TypeORM
- Docker

## Instalar e executar projeto

```yml
$ npm i
$ docker compose up -d
$ npm run typeorm migration:run
$ npm run seed:seeder
$ npm run dev
```
